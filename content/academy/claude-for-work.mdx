# 学習トラック: Claude for Work

[Claude for Work - Anthropic Learn](https://www.anthropic.com/learn/claude-for-work)

## 概要

「Claude for Work」は、**組織全体で Claude を導入し、チームの生産性を最大化する**ための学習トラックです。Claude の基本から職種別のユースケース、各種機能の活用方法、推奨コースまでを9つのトピックで体系的にまとめています。

## 9つのトピック

### 1. New to Claude? Start Here!（はじめての方はこちら）

Claude との対話で最大限の成果を引き出すための基礎を学びます。

- [claude.ai](https://claude.ai) での Claude の使い始め方
- Claude でできることのインスピレーション集
- **推奨コース**: [Claude 101](https://anthropic.skilljar.com/claude-101)

まずはここから始めて、Claude の基本的な対話の仕方とインターフェースの操作方法を身につけましょう。

### 2. Meet the Models（モデル紹介）

最新かつ最も高性能なモデル、Claude Sonnet 4.5 と Claude Opus 4.6 について学びます。

- claude.ai でモデルを試す
- Sonnet 4.5 の活用方法
- Opus 4.6 の活用方法
- 各モデル（Opus、Sonnet、Haiku）の特徴と使い分け

タスクの複雑さや速度要件に応じてモデルを選択することで、最適な結果を得られます。

### 3. Ways to Use Claude（Claude の活用方法）

チーム横断で Claude による生産性向上を実現します。

- 全ユースケースの一覧
- **エンジニアリング**: コードレビュー、デバッグ、ドキュメント作成
- **人事（HR）**: 求人票作成、面接質問の準備、ポリシードキュメント整備
- **マーケティング**: コンテンツ作成、コピーライティング、市場調査
- **プロダクトマネジメント**: PRD 作成、ユーザーストーリー、競合分析
- **営業（Sales）**: 提案書作成、メール文面、顧客対応
- **チームオフサイト計画**: イベント企画、アジェンダ作成

### 4. Artifacts（アーティファクト）

Claude との対話中にファイルやコンテンツを作成・管理・共同編集する機能です。

- Artifacts の使い方ガイド
- ドキュメント、コード、表、図などの生成
- 生成物の編集とダウンロード
- チームメンバーとの共有

コーディングの知識がなくても、視覚的なコンテンツや構造化されたドキュメントを作成できます。

### 5. Projects（プロジェクト）

作業を整理し、複雑なタスクで共同作業するための機能です。

- プロジェクトの作成と管理
- ファイルやドキュメントのアップロードによるコンテキスト設定
- チームメンバーとのプロジェクト共有
- 繰り返し使うワークフローの効率化

### 6. Skills（スキル）

詳細な指示を提供することで、特定のタスクをより正確に実行させる機能です。

- Skills の概要
- Claude での Skills の使い方
- カスタム Skills の作成方法
- チーム内での Skills テンプレート共有

### 7. Research（リサーチ）

複数のソースをエージェント的に検索し、包括的なインサイトを提供する機能です。

- リサーチモードの使い始め方
- Google Workspace との連携によるリサーチ
- Web 検索と社内情報の横断検索
- 調査レポートの自動生成

### 8. Tools & Integrations（ツールと連携）

お気に入りのツールと接続してワークフローを効率化します。

- ツール連携の設定方法
- **GitHub 連携**: コードリポジトリへのアクセス、PR レビュー
- **Google Docs 連携**: ドキュメントの読み取りと編集
- **分析ツール**: データ分析とビジュアライゼーション

## 組織導入のステップ

1. **個人での習熟**: まず Claude 101 でチームメンバー全員が基本を理解
2. **ユースケース特定**: 各部門での具体的な活用シーンを洗い出す
3. **Skills の整備**: 部門共通の Skills テンプレートを作成・共有
4. **ツール連携**: GitHub、Google Docs 等の既存ツールとの統合を設定
5. **継続的改善**: リサーチモードや新機能の導入を段階的に進める

## 対象者

- 組織で Claude の導入を推進する方
- チームの生産性向上を図りたいマネージャー
- 職種別の具体的な活用方法を知りたい方
