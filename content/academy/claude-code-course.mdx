# Claude Code in Action

[Claude Code in Action - Anthropic Academy](https://anthropic.skilljar.com/claude-code-in-action)

## コース概要

Claude Code in Action は、AI コーディングアシスタント「Claude Code」を開発ワークフローに統合するための**実践コース**です。Claude Code のアーキテクチャの理解から、実際の開発現場での活用、高度なインテグレーション戦略までを包括的にカバーします。

## 8つの主要コンピテンシー

### 1. コーディングアシスタントのアーキテクチャ

AI システムがコードベースとどのように対話するかを理解します。ツール統合の仕組みと、コード分析を可能にする技術的基盤を学びます。Claude Code が単なるチャットボットではなく、ファイルシステム、ターミナル、Git などのツールを駆使してコードを理解・操作する仕組みを理解することが重要です。

### 2. ツール利用システム

複数のツールを組み合わせて、複雑なマルチステップのプログラミングタスクに対応する方法を学びます。ファイル読み書き、コマンド実行、検索など、Claude Code が活用する各種ツールの特性と使い分けを理解します。

### 3. コンテキスト管理

関連する会話コンテキストを維持し、プロジェクトリソースを効果的に参照する方法を学びます。CLAUDE.md ファイルの設定、プロジェクト構造の提供、長い会話でのコンテキストウィンドウの最適な活用方法などを理解します。

### 4. ビジュアルコミュニケーションワークフロー

ビジュアル入力を使ってインターフェースの変更を伝える方法と、コードベース変更のための高度なプランニング機能を活用する方法を学びます。スクリーンショットやモックアップを使った指示、差分のプレビューなどのテクニックです。

### 5. カスタムオートメーション

繰り返し発生する開発タスクを効率化するための、再利用可能なコマンドとオートメーションの構築方法を学びます。カスタムスラッシュコマンド、Skills、フックなどの機能を活用します。

### 6. MCP サーバー統合

外部ツールやサービスを統合して、Claude Code の機能を拡張する方法を学びます。ブラウザオートメーション、データベース接続、専門的なワークフローなど、MCP サーバーを通じた連携パターンを理解します。

### 7. GitHub インテグレーション

自動化されたコードレビュープロセスのセットアップと、バージョン管理ワークフローへの AI アシスタンスの統合方法を学びます。プルリクエストの作成・レビュー、CI/CD パイプラインとの連携などを扱います。

### 8. 推論モード

さまざまな複雑さのプログラミング課題に対して、thinking（思考）モードや planning（計画）モードなどの推論アプローチを適用する方法を学びます。単純なタスクと複雑なリファクタリングで、どのようにモードを使い分けるかを理解します。

## 前提条件

- コマンドラインインターフェースとターミナル操作の基本的な知識
- Git を使ったバージョン管理の基本的な理解

## 対象者

- AI アシスタンスを開発ワークフローに統合したいソフトウェア開発者
- AI を活用した GitHub インテグレーションを導入したいチーム
- コーディングの生産性を向上させたいエンジニア

## Enterprise セットアップ

組織で Claude Code を導入する場合、Amazon Bedrock や Google Cloud Vertex AI を通じたエンタープライズセットアップも本コースでカバーされています。セキュリティポリシーに準拠した形での導入方法を学べます。
