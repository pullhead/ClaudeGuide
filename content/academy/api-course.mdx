# Building with the Claude API

[Building with the Claude API - Anthropic Academy](https://anthropic.skilljar.com/claude-with-the-anthropic-api)

## コース概要

Building with the Claude API は、Anthropic API を使って Claude AI をアプリケーションに統合するための**開発者向け実践コース**です。基本的な API 操作から高度なプロンプト技法、ツール統合、アーキテクチャパターンまでを体系的に学べる動画コースです。

## 学習内容

### API 基礎

- **API 認証とリクエスト設定** — API キーの取得、リクエストヘッダーの設定、基本的な API 呼び出しの方法
- **Messages API** — Claude との対話の基本となる Messages API の使い方。単一ターンおよびマルチターンの会話実装
- **システムプロンプト** — モデルの動作を制御するシステムプロンプトの設定方法

### SDK（ソフトウェア開発キット）

Anthropic は5つの言語向けに公式 SDK を提供しています。

- **Python SDK** — 最も広く使われている SDK。非同期サポートも充実
- **TypeScript SDK** — Node.js / ブラウザ環境向け
- **Java SDK** — エンタープライズ Java アプリケーション向け
- **Go SDK** — Go 言語でのネイティブ統合
- **Ruby SDK** — Ruby / Rails アプリケーション向け

### プロンプトエンジニアリング

- **XML 構造化** — XML タグを使ったプロンプトの構造化テクニック
- **Few-shot learning** — 例示ベースの学習によるモデル出力の改善
- **プロンプト評価ワークフロー** — 自動採点を用いたプロンプトの品質評価

### ストリーミング

リアルタイムにレスポンスを受け取るストリーミング機能の実装方法を学びます。ユーザー体験の向上に不可欠なテクニックです。

### ツール利用（Tool Use）

Claude に外部ツールやカスタム関数を呼び出させる機能の実装方法を学びます。

- カスタムツールの定義と統合
- ツール呼び出しの結果を会話に組み込む方法
- 複数ツールの連携パターン

### RAG（Retrieval Augmented Generation）

外部データソースを組み合わせた検索拡張生成の実装方法を学びます。

- エンベディングと検索の基礎
- ベクトルデータベースとの統合
- コンテキスチュアルリトリーバルの実装

### 拡張機能

- **画像分析（Vision）** — 画像をアップロードして Claude に分析させる方法
- **PDF 処理** — PDF ドキュメントからのテキスト抽出と分析
- **プロンプトキャッシング** — パフォーマンス最適化とコスト削減のための手法

### MCP サーバー開発

Model Context Protocol（MCP）サーバーの構築方法を学びます。Claude が外部サービスやデータソースにアクセスするための標準プロトコルです。

### エージェントシステム

- **エージェントベースのシステム設計** — 自律的にタスクを実行するエージェントの構築
- **並列化とルーティング** — 複数のタスクを効率的に処理するアーキテクチャパターン

## 前提条件

- Python プログラミングの基礎知識
- JSON データ処理の基本的な理解

## 対象者

- バックエンド / フルスタック開発者
- データエンジニア
- DevOps エンジニア
- テクニカルアーキテクト
- AI を活用したチャットボット、仮想アシスタント、コンテンツ生成ツールを構築するソフトウェアエンジニア
