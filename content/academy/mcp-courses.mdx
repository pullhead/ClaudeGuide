## Model Context Protocol（MCP）とは

Model Context Protocol（MCP）は、AI モデルが外部のツール、データソース、サービスにアクセスするための**オープンな標準プロトコル**です。Anthropic が開発し、オープンソースとして公開しています。MCP を使うことで、Claude をさまざまな外部システムと統合し、その能力を大幅に拡張できます。

Anthropic Academy では、MCP に関する2つのコースが提供されています。

---

## コース1: Introduction to Model Context Protocol（入門）

[受講する](https://anthropic.skilljar.com/introduction-to-model-context-protocol)

### 概要

MCP の基本概念とセットアップ方法を学ぶ入門コースです。MCP を初めて使う開発者を対象に、プロトコルの仕組みから実際の設定までをステップバイステップで解説します。

### 学習内容

#### MCP の基本概念

- MCP のアーキテクチャ（ホスト、クライアント、サーバーの関係）
- MCP が解決する課題（N×M 統合問題の解消）
- MCP の3つのプリミティブ（Tools、Resources、Prompts）

#### セットアップと設定

- MCP サーバーのインストールと基本設定
- Claude Desktop での MCP サーバー設定方法
- 設定ファイル（`claude_desktop_config.json`）の書き方

#### プリビルトサーバーの利用

- 既存の MCP サーバー（ファイルシステム、GitHub、Slack、データベースなど）の導入方法
- npm / uvx を使ったサーバーの起動
- トラブルシューティングの基本

#### 実践演習

- Claude Desktop から MCP サーバー経由でツールを利用する
- ローカルファイルの操作やデータベースクエリの実行

## コース2: Model Context Protocol — Advanced Topics（上級）

[受講する](https://anthropic.skilljar.com/model-context-protocol-advanced-topics)

### 概要

MCP の高度なトピックを扱う上級コースです。入門コースの知識を前提に、リモートサーバーのデプロイ、MCP Connector の活用、複雑なインテグレーションパターンを学びます。

### 学習内容

#### リモート MCP サーバー

- リモートサーバーのデプロイと運用
- OAuth 2.1 を使った認証フロー
- セキュリティのベストプラクティス

#### MCP Connector

- MCP Connector を使った Messages API との統合
- サーバーサイドアプリケーションからの MCP 利用
- プログラマティックな MCP サーバーアクセス

#### 高度なパターン

- 複数の MCP サーバーの連携
- カスタム MCP サーバーの開発
- エンタープライズ環境での MCP デプロイメント
- パフォーマンス最適化とエラーハンドリング

#### 実世界のユースケース

- 社内ツールとの統合事例
- データパイプラインへの MCP 組み込み
- マルチエージェントシステムでの MCP 活用

## 前提条件

- **入門コース**: プログラミングの基礎知識、コマンドラインの基本操作
- **上級コース**: 入門コースの修了、Web API / サーバー開発の基礎知識

## 推奨学習順序

1. まず入門コースで MCP の基本を理解する
2. Claude Desktop で実際に MCP サーバーを設定・利用してみる
3. 上級コースでリモートサーバーや高度なパターンを学ぶ
4. 自分のプロジェクトでカスタム MCP サーバーを開発する
