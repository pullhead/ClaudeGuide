# Introduction to Agent Skills

[Introduction to Agent Skills - Anthropic Academy](https://anthropic.skilljar.com/introduction-to-agent-skills)

## コース概要

Introduction to Agent Skills は、Claude Code における **Skills 機能**の構築・設定・配布方法を学ぶコースです。「同じことを繰り返し説明するのをやめて、Claude に一度教えれば済むようにする」をコンセプトに、Skills の概念から実践的な実装までをカバーします。

Skills は、CLAUDE.md、フック、サブエージェントなどの他のカスタマイズオプションとどう異なるかを理解し、適切に使い分けることが重要です。

## 学習内容

### Skills の基礎

- **Skills とは何か** — マークダウンベースの再利用可能な指示セット。特定のタスクに対して Claude が自動的に適用する
- **他のカスタマイズとの違い** — CLAUDE.md（プロジェクト全体の設定）、フック（イベントトリガー）、サブエージェント（タスク委任）との使い分け
- **Skills が解決する課題** — 繰り返しの指示、一貫性のない出力、コンテキスト効率の低下

### Skills の作成

- **SKILL.md フロントマター** — Skills ファイルのメタデータ記述方法（名前、説明、トリガー条件）
- **効果的な description の書き方** — Claude が確実にトリガーマッチングできる説明文の作成テクニック
- **スキルディレクトリの整理** — progressive disclosure（段階的開示）を活用した効率的なファイル構成

### 高度な設定

- **allowed-tools によるツールアクセス制限** — Skills が利用できるツールを明示的に制限する方法
- **サブエージェントへの接続** — Skills をカスタムサブエージェントに接続してタスクを委任する方法
- **条件付きトリガー** — 特定の状況でのみ Skills を起動させる設定

### Skills の配布と共有

- **リポジトリコミットによる共有** — プロジェクトリポジトリに Skills を含めてチーム共有
- **プラグインとしての配布** — Skills を Claude Code プラグインとしてパッケージ化
- **エンタープライズ管理設定** — 組織全体への Skills の一括デプロイ

### トラブルシューティング

- Skills がトリガーされない場合の対処法
- ランタイムエラーの診断と修正
- デバッグツールの活用

## API と Claude Code での利用

Skills は以下の2つの方法で利用できます。

### Claude Code での利用

```
# Skills を含むディレクトリ構造の例
.claude/
  skills/
    code-review.md
    test-writing.md
    commit-message.md
```

Claude Code はタスクの内容に基づいて自動的に適切な Skills を選択・適用します。`/スキル名` でスラッシュコマンドとしても呼び出せます。

### API での利用

API 経由で Skills を利用する場合、システムプロンプトに Skills の内容を含めることで同様の効果を得られます。

## ベストプラクティス

- **1つの Skills = 1つのタスク** — 各 Skills は明確に定義された1つのタスクに焦点を当てる
- **具体的な description** — 曖昧な説明ではなく、トリガー条件が明確にわかる記述にする
- **テスト可能な出力** — Skills の結果が検証可能な形式になるよう設計する
- **段階的な構築** — シンプルな Skills から始めて、徐々に複雑さを追加する

## 最終ゴール

コース修了時には、Skills ベースの完全なワークフローを構築できるようになります。Claude を一貫性があり、コンテキスト効率が高く、チームの基準に沿った状態に保つためのスキルを身につけます。

## 対象者

- Claude Code を使っている開発者
- チーム内で Claude の動作を標準化したい方
- 繰り返しタスクの自動化に興味がある方
