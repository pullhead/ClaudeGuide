### 2. APIs & SDKs

Anthropic の API、SDK、その他の開発ツールについて学びます。Claude アプリケーション構築の基盤となるトピックです。

#### SDK（ソフトウェア開発キット）

5つのプログラミング言語に対応した公式 SDK が用意されています:

| SDK | 言語 | 用途 |
|-----|------|------|
| Python SDK | Python | データサイエンス、バックエンド、スクリプティング |
| TypeScript SDK | TypeScript/JavaScript | Web アプリケーション、Node.js |
| Java SDK | Java | エンタープライズアプリケーション |
| Go SDK | Go | マイクロサービス、インフラツール |
| Ruby SDK | Ruby | Web アプリケーション（Rails 等） |

#### API の基本

- **Messages API** — Claude との対話の基本 API。テキスト・画像の入力、ストリーミング応答に対応
- **Message Batches API** — 複数の Messages API リクエストを一括処理。大量処理時のコスト最適化に有効
- **プロンプトキャッシング** — 繰り返し使用するプロンプトをキャッシュして API 利用を最適化
- **Amazon Bedrock API** — AWS 上で Claude を利用するための統合ガイド
- **Vertex AI API** — Google Cloud 上で Claude を利用するための統合ガイド

#### 高度な API

- **Admin API** — ワークスペースの権限管理と組織設定
- **Files API** — ファイルのアップロードと管理
- **PDF サポート** — PDF からのテキスト抽出とビジュアルコンテンツの理解
- **[実験的 API] プロンプト生成** — 適切に構造化されたプロンプトを自動生成
- **[実験的 API] プロンプト改善** — コード内で直接プロンプトを改善
- **[実験的 API] プロンプトテンプレート化** — 変数を自動抽出してプロンプトをテンプレート化

## このトラックの活用方法

### 学習パス（推奨順序）

1. **初めての方**: まず「APIs & SDKs」で基本的な API 呼び出しを学び、[Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api) コースを受講
2. **ツール統合**: 「Tool Use」「MCP」で外部ツール連携の基本を習得
3. **アプリ開発**: 「RAG」「Agents」「Extended Thinking」で実践的な機能を実装
4. **品質向上**: 「Prompt Engineering」「Evaluations」で出力品質を体系的に最適化
5. **本番運用**: 「Prompt Caching」でコスト最適化、「Claude 4.5 Models」でモデル選択を最適化
6. **高度な機能**: 「Vision」「Computer Use」「Skills」で応用領域を拡大
