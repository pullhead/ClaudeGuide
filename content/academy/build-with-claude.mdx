# 学習トラック: Build with Claude

[Build with Claude - Anthropic Learn](https://www.anthropic.com/learn/build-with-claude)

## 概要

「Build with Claude」は、Claude を使ったアプリケーション開発のための**開発者向け学習トラック**です。API ガイド、ベストプラクティス、チュートリアル、コースなど、包括的なリソースを提供しています。15のトピックにわたり、開発に必要な知識を体系的に学べます。

## 15のトピック

### 1. Claude 4.5 Models

最新の Claude モデルの機能とインテリジェンスについて学びます。

- Sonnet 4.5 の新機能
- Claude 4 へのモデルマイグレーションガイド
- モデル選択基準と比較
- プロンプト最適化のベストプラクティス

### 2. APIs & SDKs

Claude アプリケーション構築のための開発ツール群。

- **SDK**: Python、TypeScript、Java、Go、Ruby の5言語対応
- **API**: Messages API、バッチ処理、プロンプトキャッシング
- **クラウド統合**: Amazon Bedrock、Google Vertex AI
- **高度な API**: Admin API、Files API、PDF テキスト抽出
- **コース**: [Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api)

### 3. Agents

複雑なタスクを理解・計画・実行する自律型エージェントの構築。

- エージェント設計パターン（Anthropic Cookbook）
- JSON 出力の実装による信頼性の高い制御
- MCP ドキュメント
- 効果的なエージェント構築ガイド

### 4. Skills

詳細なタスク指示で Claude のパフォーマンスを向上。

- Skills の概要とフレームワーク
- Skills 作成のベストプラクティス
- API 統合方法
- Claude Code との統合
- **コース**: [Introduction to Agent Skills](https://anthropic.skilljar.com/introduction-to-agent-skills)

### 5. Model Context Protocol（MCP）

高度なアプリケーション開発のための標準プロトコル。

- Claude Desktop での統合
- プリビルト MCP サーバー
- リモート MCP サーバーのデプロイ
- Messages API との接続
- **コース**: [入門](https://anthropic.skilljar.com/introduction-to-model-context-protocol) / [上級](https://anthropic.skilljar.com/model-context-protocol-advanced-topics)

### 6. Claude Code

AI アシスタントによる開発環境の加速。

- インストールと IDE 統合
- Google Vertex / Amazon Bedrock エンタープライズセットアップ
- 一般的なワークフローテンプレート
- トラブルシューティングガイド
- **コース**: [Claude Code in Action](https://anthropic.skilljar.com/claude-code-in-action)

### 7. Tool Use

外部統合で Claude の機能を拡張。

- コード実行、テキスト編集、Web 検索
- インタラクティブなツール利用コース
- 実装クックブック

### 8. Extended Thinking

複雑な問題解決のための Claude の推論能力の強化。

- 拡張思考モデルの概要
- 実装のベストプラクティス
- 数学、分析、コーディングなどでの活用

### 9. RAG（Retrieval Augmented Generation）

外部データソースと Claude を組み合わせたシステムの構築。

- カスタマーサポートエージェントテンプレート
- Voyage AI エンベディング統合
- LlamaIndex / MongoDB との統合
- コンテキスチュアルリトリーバル

### 10. Prompt Engineering

効果的なプロンプティングによる Claude のパフォーマンス最適化。

- インタラクティブなプロンプトエンジニアリングチュートリアル
- 実世界シナリオの練習
- プロンプトジェネレーターツール
- 包括的なドキュメント

### 11. Evaluations

Claude のパフォーマンスを体系的にテスト・検証。

- 評価フレームワークの構築
- 自動評価テクニック
- Claude Console の Eval ツール

### 12. Prompt Caching

パフォーマンスの最適化と運用コストの削減。

- 実装ガイド（Anthropic Cookbook）
- キャッシング戦略と設計パターン

### 13. Vision

Claude の画像理解能力の活用。

- ビジョンの基礎とベストプラクティス
- 画像からのテキスト抽出
- チャート・グラフの分析

### 14. Computer Use

Claude にデスクトップ環境を操作させる機能。

- クイックスタートデモ
- Anthropic API との統合
- データ処理ドキュメント

### 15. Hackathon Hacker Guide

ハッカソン参加者向けのクイックスタートリソース。

- 開発者アカウントのセットアップと API キーの取得
- 最初の API 呼び出し
- Claude Code のインストール
- Anthropic Cookbook のコード例
- クイックスタートリポジトリ

## このトラックの活用方法

1. **初めての方**: まず「APIs & SDKs」で基本的な API 呼び出しを学ぶ
2. **アプリ開発**: 「Tool Use」「RAG」「Agents」で実践的な機能を実装
3. **品質向上**: 「Prompt Engineering」「Evaluations」で出力品質を最適化
4. **本番運用**: 「Prompt Caching」でコスト最適化、クラウド統合で本番デプロイ
