# Anthropic Academy 概要

[Anthropic Academy](https://anthropic.skilljar.com/) | [Learn](https://www.anthropic.com/learn)

## Anthropic Academy とは

Anthropic Academy は、Anthropic が提供する**無料の公式学習プラットフォーム**です。Claude の基本的な使い方から API を使った開発、組織導入まで、幅広いトピックをカバーする体系的なコースが用意されています。すべてのコースは無料で受講でき、修了時には**修了証（Certificate）**を取得できます。

プラットフォームは Skilljar 上でホストされており、動画レッスン、インタラクティブな演習、クイズなどを通じて実践的に学習できます。

## 3つの学習トラック

Anthropic は学習目的別に3つのトラックを用意しています。各トラックは独立していますが、必要に応じて横断的に学習できます。

### 1. Build with Claude（開発者向け）

Claude を使ったアプリケーション開発のための包括的なガイドとベストプラクティスを提供します。**15のトピック**をカバー:

- **モデル & 基盤**: Claude 4.5 Models、APIs & SDKs
- **エージェント & 拡張**: Agents、Skills、MCP、Claude Code、Tool Use
- **高度な機能**: Extended Thinking、RAG、Vision、Computer Use
- **最適化**: Prompt Engineering、Evaluations、Prompt Caching
- **クイックスタート**: Hackathon Hacker Guide

主なゴール: API を使った Claude アプリの構築、エージェントシステムの設計、本番環境でのパフォーマンス最適化

> [詳細: Build with Claude](/academy/build-with-claude)

### 2. Claude for Work（組織・チーム向け）

組織全体での Claude 導入と生産性最大化のための学習トラックです。**9つのトピック**をカバー:

- **入門**: Start Here、Meet the Models
- **活用方法**: Ways to Use Claude（職種別ユースケース: エンジニアリング、HR、マーケティング、PM、営業等）
- **機能**: Artifacts、Projects、Skills、Research
- **連携 & 学習**: Tools & Integrations、Courses

主なゴール: チーム全体での Claude 導入、職種別のベストプラクティス確立、ワークフロー効率化

> [詳細: Claude for Work](/academy/claude-for-work)

### 3. Claude for You（個人向け）

個人のプロジェクトや日常タスクで Claude を活用するための学習トラックです。

- **基礎**: AI Fluency: Framework & Foundations コースによる AI リテラシーの習得
- **モデル理解**: Meet the Models で用途に応じたモデル選択
- **機能活用**: Artifacts、Projects、Skills、Tools & Integrations
- **教育**: 教育者・学生向けの専門コース

主なゴール: AI リテラシーの獲得、日常タスクでの Claude 活用、個人の生産性向上

> [詳細: Claude for You](/academy/claude-for-you)

## 修了証について

すべてのコースで、最終アセスメントに合格すると**無料の修了証**が発行されます。修了証は LinkedIn プロフィールなどに掲載でき、Claude に関する知識・スキルの証明として活用できます。

## 始め方

1. [Anthropic Academy](https://anthropic.skilljar.com/) にアクセス
2. 無料アカウントを作成（メールアドレスのみ）
3. 上記の「学習目的別おすすめコース」を参考に、興味のあるコースを選択して受講開始
4. コース修了後、修了証を取得
