# Anthropic Academy 概要

[Anthropic Academy](https://anthropic.skilljar.com/) | [Learn](https://www.anthropic.com/learn)

## Anthropic Academy とは

Anthropic Academy は、Anthropic が提供する**無料の公式学習プラットフォーム**です。Claude の基本的な使い方から API を使った開発、組織導入まで、幅広いトピックをカバーする体系的なコースが用意されています。すべてのコースは無料で受講でき、修了時には**修了証（Certificate）**を取得できます。

プラットフォームは Skilljar 上でホストされており、動画レッスン、インタラクティブな演習、クイズなどを通じて実践的に学習できます。

## 3つの学習トラック

Anthropic は学習目的別に3つのトラックを用意しています。

### 1. Build with Claude（開発者向け）

Claude を使ったアプリケーション開発のための包括的なガイドとベストプラクティスを提供します。API・SDK、エージェント、MCP、ツール利用、拡張思考、RAG、プロンプトエンジニアリングなど15のトピックをカバーします。

→ [詳細: Build with Claude](/academy/build-with-claude)

### 2. Claude for Work（組織・チーム向け）

組織全体での Claude 導入と生産性最大化のための学習トラックです。職種別のユースケース、Artifacts、Projects、Skills、リサーチ、ツール連携などを扱います。

→ [詳細: Claude for Work](/academy/claude-for-work)

### 3. Claude for You（個人向け）

個人のプロジェクトや日常タスクで Claude を活用するための学習トラックです。AI Fluency コースを中心に、モデルの理解、各機能の活用方法を学びます。

→ [詳細: Claude for You](/academy/claude-for-you)

## 修了証について

すべてのコースで、最終アセスメントに合格すると**無料の修了証**が発行されます。修了証は LinkedIn プロフィールなどに掲載でき、Claude に関する知識・スキルの証明として活用できます。

## 始め方

1. [Anthropic Academy](https://anthropic.skilljar.com/) にアクセス
2. 無料アカウントを作成（メールアドレスのみ）
3. 興味のあるコースを選択して受講開始
4. コース修了後、修了証を取得
