# Everything Claude Code

[Everything Claude Code](https://github.com/affaan-m/everything-claude-code)

Claude Codeを最大限に活用するための包括的なガイドです。公式ドキュメントとコミュニティの知見を統合し、セットアップから高度なワークフローまでを日本語で解説します。

## このセクションの内容

### [セットアップ](/everything-claude-code/getting-started)
インストール方法、初期設定、基本的なコマンド操作を解説します。macOS/Linux/Windowsでの導入手順から、最初の対話までをカバーします。

### [CLAUDE.md](/everything-claude-code/claude-md)
プロジェクト固有の指示ファイル `CLAUDE.md` の書き方と階層構造を解説します。プロジェクトレベル・ユーザーレベルの設定を使い分ける方法を学びます。

### [コマンド & スキル](/everything-claude-code/commands-skills)
スラッシュコマンドとカスタムスキルの定義方法を解説します。57以上のコマンドと108のスキルを活用してワークフローを構築する方法を紹介します。

### [フック](/everything-claude-code/hooks)
フックシステムによるツールイベントの自動化を解説します。セキュリティフック、カスタマイズフック、フックの設定方法を学びます。

### [サブエージェント](/everything-claude-code/subagents)
Agentツールを使った並列処理とworktreeの活用方法を解説します。マルチエージェントオーケストレーションで複雑なタスクを分割する手法を紹介します。

### [MCPサーバー](/everything-claude-code/mcp-servers)
Model Context Protocol（MCP）サーバーの設定と活用方法を解説します。GitHub、Supabase、Vercelなど外部サービスとの連携方法を学びます。

### [トークン最適化](/everything-claude-code/token-optimization)
コンテキストウィンドウの管理とコスト最適化の手法を解説します。モデル選択戦略、戦略的コンパクション、MCPの最適化でコストを削減します。

### [ワークフロー](/everything-claude-code/workflows)
開発ワークフローの自動化とオーケストレーション手法を解説します。新機能開発、バグ修正、リリース準備など実践的なパターンを紹介します。

### [Tips & Tricks](/everything-claude-code/tips)
Claude Codeを効果的に使うための実践的なTipsを集めました。日常の開発で役立つテクニックとベストプラクティスを紹介します。

## ソースについて

このセクションの内容は、以下のソースを基に構成しています：

- [Claude Code 公式リポジトリ](https://github.com/anthropics/claude-code)
- [Everything Claude Code](https://github.com/affaan-m/everything-claude-code) — Anthropicハッカソン優勝プロジェクト。25以上のエージェント、108のスキル、57のコマンドを提供する包括的な最適化システムです。
