# 12週間学習カリキュラム

> 原文: [How to Become a Claude Certified Architect in 12 Weeks](https://x.com/heynavtoor/status/2035787873665085543) — Nav Toor (@heynavtoor)

## Claude Certified Architect とは

Claude Certified Architect, Foundations（CCA）は、Anthropic初の公式技術認定資格である。プロンプティングのバッジではない。コース修了証でもない。Claudeを使って本番グレードのAIシステムを設計・出荷できるかどうかを問う、監督付き60問の試験である。

試験は5つのドメインをカバーする:

| ドメイン | 内容 | 配分 |
|---------|------|------|
| D1 | エージェントアーキテクチャとオーケストレーション | 27% |
| D2 | ツール設計とMCP統合 | 18% |
| D3 | Claude Codeの設定とワークフロー | 20% |
| D4 | プロンプトエンジニアリングと構造化出力 | 20% |
| D5 | コンテキスト管理と信頼性 | 15% |

- **合格ライン**: 1,000点満点中720点
- **問題形式**: シナリオベース — 実際の本番環境に即した状況に置かれ、正しいアーキテクチャ上の判断を求められる。誤答はランダムではなく、本番環境での影響を理解する前にエンジニアが手を伸ばしがちな典型的なアンチパターンが並ぶ
- **受験料**: 1回$99
- **Partner Network**: 加入無料。最初の5,000名のパートナー企業従業員は早期アクセスを無償で取得可能

## なぜ今この資格が重要なのか

3つのことが同時に起きている。

**エンタープライズ導入の爆発的拡大。** Fortune 100企業の70%がClaudeを使用している。Accentureは30,000人のプロフェッショナルを育成中。Cognizantは350,000人の従業員にClaudeアクセスを開放した。DeloitteとInfosysはアンカーパートナーである。これらの企業は実験しているのではない。デプロイしている。そしてシステムを設計する認定アーキテクトが必要なのだ。

**求人市場の変化。** AIエンジニアの求人は前年比143%増。米国の平均年収は$175,000。しかし企業が求めているのはAIとチャットできる人材ではない。AIシステムを設計できる人材だ。CCA資格は前者ではなく後者ができることを証明する。

**ファーストムーバーの窓が閉じつつある。** AWSクラウド認定は、求人票で「任意」から「必須」になるまで5年かかった。AIの進化速度を考えれば、Claude認定は同じ道のりを年単位ではなく月単位で圧縮するだろう。今認定を取得したエンジニアは、認定が当たり前になる前にクレディビリティを確立できる。

## 12週間ロードマップ（すべてのリソースが無料）

### Week 1-2: 基礎

**目標**: Claudeとのコラボレーション原則のベースライン知識を構築する。

**コース1: [Claude 101](https://anthropic.skilljar.com/claude-101)**（Anthropic Academy、無料）

Claudeの利用パターン、機能、プロジェクト、アーティファクト、スキル、コネクタ、実用的なワークフローをカバーする。数ヶ月Claudeを使っていたとしてもここからスタートすること。試験は全体を通じてこのベースライン知識を前提としている。

**コース2: [AI Fluency: Framework and Foundations](https://anthropic.skilljar.com/ai-fluency-framework-foundations)**（Anthropic Academy、無料）

大学教授と共同開発。効果的なAIコラボレーションの原則をカバー: 委任（Delegation）、記述（Description）、識別（Discernment）、勤勉（Diligence）。人間とAIがどう協働するかの概念的フレームワーク。

**マイルストーン**: Claudeにできることとできないことを説明でき、Chat・Cowork・Codeの違いを理解し、責任あるAI委任の原則を明確に述べられる。

---

### Week 3-5: Claude APIで構築する

**目標**: 試験の基幹となるコア技術マテリアルをマスターする。

**コース3: [Building Applications with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api)**（Anthropic Academy、無料、8.1時間）

フラッグシップコース。基本的なAPIリクエストから高度なエージェントアーキテクチャとRAGパイプラインまですべてをカバーする。システムプロンプト、ツール使用、ストリーミング、認証、構造化出力、コンテキストウィンドウ管理、本番アプリケーションパターン。この1コースだけで試験のドメイン1、4、5に対応する。

**補足**: [Anthropic APIドキュメント](https://docs.anthropic.com/)（無料）。コースと並行して公式ドキュメントを読むこと。試験が問うのは理論ではなく本番環境での判断だ。ドキュメントは実際にどう動くかを示している。

**マイルストーン**: ツール呼び出し、構造化出力を使用し、エラーを適切に処理する動くClaudeアプリケーションを構築できる。エージェントループを理解し、シングルエージェントとマルチエージェントアーキテクチャのどちらを使うべきか説明できる。

---

### Week 6-7: Model Context Protocol（MCP）

**目標**: ドメイン2「ツール設計とMCP統合」をマスターする。

**コース4: [Introduction to Model Context Protocol](https://anthropic.skilljar.com/introduction-to-model-context-protocol)**（Anthropic Academy、無料）

MCPの基礎をカバー: ツール、リソース、プロンプト。ツール設計とMCP統合の試験ドメインに直接対応する。

**コース5: [Model Context Protocol Advanced](https://anthropic.skilljar.com/model-context-protocol-advanced-topics)**（Anthropic Academy、無料）

Pythonを使ったMCPサーバーとクライアントの構築を深掘りする。サンプリング、通知、ファイルシステムアクセス、本番MCPサーバー開発のためのトランスポートメカニズム。

**補足**: [MCPドキュメント](https://modelcontextprotocol.io/)（無料）。オープン仕様。コースと並行して読むこと。MCPはAIを外部ツールに接続するユニバーサルスタンダードになりつつある — AIにとってのUSB-Cのようなものだ。

**マイルストーン**: MCPツールを設計・構築でき、類似するツールを明確に区別するツール説明を書き、エラーカテゴリとリトライ可能フラグを含む構造化エラーレスポンスを実装できる。

---

### Week 8-9: Claude Code

**目標**: ドメイン3「Claude Codeの設定とワークフロー」をマスターする。

**コース6: [Claude Code in Action](https://anthropic.skilljar.com/claude-code-in-action)**（Anthropic Academy、無料）

Claude Codeの設定、開発者ワークフロー、CLAUDE.md階層、パス固有ルール、カスタムスラッシュコマンド、スキル、フック、CI/CDパイプラインへの統合をカバーする。

**コース7: Claude Code on Coursera**（Vanderbilt大学、無料で聴講可能）

Generative AI Software Engineering SpecializationのパートでJules White博士が担当。AIファースト開発とBest of Nパターンをカバー。Claude Codeワークフローに別の視点を提供する。

**コース8: [Building Skills for Claude Code](https://anthropic.skilljar.com/introduction-to-agent-skills)**（Anthropic Academy、無料）

スキルの作成、設定、配布方法。再利用可能なワークフローと本番グレードのスキルアーキテクチャに関する試験のフォーカスに対応する。

**マイルストーン**: 実プロジェクトのCLAUDE.md階層をパススコープルールで設定でき、カスタムスキルを構築し、Claude Codeを開発パイプラインに統合できる。

---

### Week 10-11: 演習とシナリオ

**目標**: 試験形式のシナリオで練習し、弱点を特定する。

**リソース: 公式CCA試験ガイドPDF**（無料）。コミュニティではこれ自体が独立した教材として機能すると評価されている。試験のブループリント、サンプル問題、直面する6つのシナリオコンテキストが含まれる。徹底的に勉強すること。

**リソース: [Claude Certifications 模擬問題](https://claudecertifications.com/)**（無料）。実際の試験と同じ構造の25問の練習問題。各問題を解き、正解がなぜ正しいのか、誤答がなぜアンチパターンを表すのかを理解すること。

**リソース: Architect Cert MCPサーバー**（無料、オープンソース）。Claudeをパーソナル試験チューターに変えるMCPサーバー。各ドメインの問題を出題し、詳細な解説を提供する。

**演習課題**: Claude Agent SDKを使って、ツール呼び出し、構造化エラーハンドリング、マルチエージェントのサブエージェントオーケストレーションを備えたエンドツーエンドのエージェントループを構築する。この1つの演習で5つの試験ドメインすべてに触れることができる。

**マイルストーン**: サンプル問題を見て、正解とそれぞれの誤答が表すアンチパターンの両方を即座に特定できる。

---

### Week 12: 最終レビューと受験

**目標**: アンチパターンを復習し、受験する。

この週は新しい教材を勉強しないこと。すでに学んだことを復習する。特にアンチパターン — 本番Claudeシステムを構築する際にエンジニアが犯しがちな典型的なミスに集中すること。試験に合格したコミュニティメンバーは一貫して、多くの誤答がランダムではなく、「正しく見えるが本番で壊れる」選択肢だと報告している。

**重点的に復習すべきアンチパターン**:
- プログラマティックな制御の代わりにシステムプロンプトにセキュリティを依存する
- エージェントのパーミッションを広げすぎる
- コンテキストウィンドウの制限を無視する
- 専門化されたサブエージェントに分解する代わりに1つの巨大なエージェントを使う
- 構造化出力のバリデーション・リトライループを省略する

準備ができたら、Anthropic Partner Networkポータルから試験を予約する。プロクター付きのオンライン試験で、自宅から受験可能。結果は即時通知される。

## 合格後

CCA Foundations資格が取得できる。雇用主、クライアント、パートナーに対して、本番グレードのClaudeシステムを設計できることを証明するものだ。

**キャリアへの影響**: CCAはすでに求人票に登場している。大手コンサルティングファームはClaude関連のデリバリーロールで必須としている。2026年を通じてエンタープライズ導入が加速する中、この資格を早期に取得することで、後から取得する数十万人のプロフェッショナルに先行できる。

**資格スタック**: Anthropicは2026年後半にセラー、デベロッパー、アドバンスドアーキテクト向けの追加認定を確認している。CCA Foundations保持者は各新ティアへの優先アクセスが与えられる。これは今後大きく成長する資格スタックの第一歩だ。

**コミュニティ**: Claude Partner Networkにより、専用技術サポート、共同マーケティング機会、そしてエンタープライズが実装パートナーを探す際に検索するディレクトリへのアクセスが得られる。ネットワークはAnthropicの1億ドル投資に支えられている。
